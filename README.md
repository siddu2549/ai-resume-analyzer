# 🚀 ResumeIQ – AI Resume Analyzer

ResumeIQ is an AI-powered web application that analyzes resumes and provides insights such as **ATS compatibility, detected skills, and improvement suggestions**.  
It helps job seekers improve their resumes to increase their chances of getting shortlisted by recruiters.

## ✨ Features

✔ Upload PDF resumes  
✔ Extract resume text automatically  
✔ Detect technical skills from resumes  
✔ Calculate resume quality score  
✔ ATS compatibility score  
✔ Impact score  
✔ Provide improvement suggestions  
✔ Modern AI-style dashboard UI

## 📊 How It Works

1️⃣ User uploads a resume (PDF)  
2️⃣ Backend extracts text from the resume  
3️⃣ NLP module detects skills  
4️⃣ Resume scoring algorithm calculates quality score  
5️⃣ Suggestions are generated for improvement  
6️⃣ Results are displayed in an interactive dashboard
